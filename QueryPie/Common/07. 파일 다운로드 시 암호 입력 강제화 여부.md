# [QueryPie] 파일 다운로드 시 암호 입력 강제화 여부

## Subscription 
Default

## Menu 
Admin > General > Company Management > Security > Others

쿼리파이 시스템에서 파일 다운로드 시 파일 암호 입력을 강제화하는 옵션 활성화 설정을 검토합니다.

**검토 대상 항목 및 예시 설정값**

![Others](images/others.png)
- `Export a file with Encryption` : Required
