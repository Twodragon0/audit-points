# [Jenkins] 14. 예외처리된 보안 알람 설정 여부

## Menu 
Dashboard > Manage Jenkins > Security

## 점검 방법 
Jenkins 업데이트 사이트에서 제공하는 보안 알람에 대해 숨겨진 알람이 있는지 **Hidden security warnings** 항목에서 검토합니다.
